## x86

CISC (Complex Instruction Set Computer) architecture. Compatability since 1978. Developed initally by Intel, used also in many AMD chips.

## ARM

RISC (Reduced Instruction Set Computer) architecture developed by Acorn (ARM = Acorn RISC Machine initially). Usually more power-efficient.

## ISA

Instruction set architecture. Specifies the instructions but not the implementation.
